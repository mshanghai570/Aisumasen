# Aisumasen

A custom, underground Reddit client for iOS inspired by John Lennon's "Mind Games" album aesthetic.

## Features
- Vintage 1970s vinyl-inspired UI
- Reddit API integration
- Custom color palette based on classic album artwork
- Native iOS performance

## Requirements
- iOS 14.0+
- Xcode 12.0+
- Swift 5.3+

## Installation
1. Clone the repository
2. Open `Aisumasen.xcodeproj` in Xcode
3. Configure your Reddit API credentials
4. Build and run

## Reddit API Setup
1. Create a Reddit app at https://www.reddit.com/prefs/apps
2. Set app type to "script"
3. Update credentials in the authentication service
